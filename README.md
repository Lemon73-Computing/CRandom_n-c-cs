キャラクターの設定を自動で決定するソフトウェアです。

### 技術
*n-c-cs: .NET Console C#*
- 言語: C# (.NET8.0)
- 形式: CUI (コンソール版)
- ベース: SkyColors (n-c-cs4)

### 対応
- OS: Windows / Mac / Linux
- 言語: 日本語

> [!NOTE]
> Macは試験動作を行っていないので、正常に動作するかわかりません。
