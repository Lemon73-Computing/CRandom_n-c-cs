# CRondom
## CRandom とは…
LECが開発したキャラクター設定自動決定プログラムである。

### 技術仕様
    言語:C#
    フレームワーク:.NET6.0/7.0
    形式:CUI(コンソール版)
    ベース:SorairoColors
    対応OS:Windows
注:(フレームワーク的にはMac/Linuxも対応可能だが、ビルドや試験動作ができないため現時点では非対応とする)<br>

### ソフトウェア開発環境
    OS:Windows10 22H2
    CPU:Intel Core i5 6400T
    RAM:DDR4 2400MHz 4GBx2枚

### 開発で利用しているソフトウェア
    IDE:Visual Studio
    バージョン管理:Git
    オープンソース化:GitHub
